Software

  • Arch Linux is my main OS. I have found it very stable and I adore the package manager.
  • I also like Kiss Linux and Gentoo, but I find them impractical at times so I use them less.
  • I have a Windows partition on my desktop for software and game compatability.
  • zsh - I just like the better tab complete, no complex config or anything
  • nvim - I like it for simple editing and writing notes For more complex stuff I use Jetbrains software
  • Qtile - Python config file is very nice and allows for lots of customisation
  • Hikari - Simple wayland floating window manager, I like the workflow and grouping systems
